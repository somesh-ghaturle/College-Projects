Computer Labs facility

Home - Admission


There are 27 Computer Labs in Campus with machines having the latest configurations. The entire labs have the best infrastructure with the state-of-the art technology. The labs have network facility, connecting the latest Intel microprocessor based computers. The labs have Pentium IV dual core machines, based nodes each with 2 GB RAM, 250 GB local hard disk, DELL color monitors, connected to through servers. Laser Printers, Data Projector, Scanner, and DVD Writers, other peripherals are connected to the network.
