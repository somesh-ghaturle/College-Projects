B.E. I Year Admission

  1. Passport size Photographs (03 Nos.)
  2. Allotment Letter (Please write User ID & Password on the back side)
  3. Score Card MH-CET 2018 / JEE MAIN 2018 Paper-I
  4. H.S.C. Mark sheet
  5. S.S.C. Mark sheet
  6. S.S.C. Passing Certificate
  7. School Leaving Certificate
  8. Nationality Certificate (if Nationality is not mentioned in Leaving Certificate)
  9. Domicile Certificate (If applicable)
  10. Income Certificate (If applicable)
  11. Caste Certificate (If applicable)
  12. Caste Validity Certificate (If applicable)
  13. Non Creamy Layer Certificate (for OBC,NT,SBC valid upto 31/03/2018
  14. Migration Certificate (Other than M.S. Board) (If applicable)
  15. Eligibility Certificate (Other than M.S. Board) (If applicable)
  16. GAP Certificate (If applicable)
  17. Affidavit for change in name / Marriage cert. / Govt. Gazette (If applicable)
  18. Physically Handicapped Certificate (If applicable)
  19. Defence Certificate “C” Form (If applicable)
  20. E.B.C. Certificate (If applicable)
  21. For Reservation Category Students : Candidate has to open Bank Account in any Nationalized Bank and submit Photocopy of first page of passbook (where Name and Account Number, IFSC Code is printed)

  5. Diploma Marksheets (All Years / Semesters)
  6. Aadhar Card
  7. Diploma Certificate

  14. Migration Certificate
  15. Eligibility Certificate

  17. Two Photocopies of above Documents
  18. Physically Handicapped Certificate [If physically handicapped]
